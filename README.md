## Overview
A modern, real-time blockchain explorer for HomeChain PoW. Optimized for V2 state persistence and high-accuracy data representation.

## New Features (v2.11)
- **Deep Data Parity**: Powered by the Blockchain-Authoritative Sync engine.
- **Smart Truncation**: Enhanced address and hash display with one-click copy.
- **Dynamic Stats**: Real-time supply, block height, and difficulty tracking.
- **Rich Details**: Pagination and deep linking for Blocks, Transactions, and Holders.

## Tech Stack
- **Frontend**: Vite + React + Vanilla CSS (Glassmorphism).
- **Backend**: Supabase (PostgreSQL).
- **Deployment**: Vercel.
